# Multi-DOF Robotic Arm Design (SolidWorks)

This repository contains the 3D computer-aided design (CAD) files for an articulated multi-DOF robotic arm, modeled using SolidWorks. The project focuses on geometric modeling, joint constraints, and mechanical assembly design tailored for robotics applications.

---

## ⚙️ Design Features
* **Degrees of Freedom (DOF):** 3 
* **Joint Types:** Revolute joints for anthropomorphic movement.
* **Application Focus:** Kinematic simulation, pick-and-place automation, and structural design.

---

## 📁 Repository Structure
* `/Assembly`: Contains the main SolidWorks Assembly file (`.SLDASM`).
* `/Parts`: Contains all individual component part files (`.SLDPRT`) used in the assembly.

---

## 🚀 How to View and Use
1. Clone or download this repository to your local machine.
2. Open the `/Assembly` folder.
3. Load the primary assembly file in **SolidWorks**.

*Note: To prevent broken file references within SolidWorks, please ensure that the `/Parts` and `/Assembly` folders remain in the same relative directory structure when downloaded.*
